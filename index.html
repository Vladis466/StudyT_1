Interacting with the API:

There are three seperate types of data you can access. These are the TimeStudies, TimeStudy Instances, and Procedures. For each of these you can choose to either see the current existing data or you can add TimeStudies and Procedures to the list of existing studies. Additionally, deleting a study will delete all associated procedures with it, so be careful. However, you cannot singularily delete a procedure. This was done in order to add rigidity to the process and is critical for the standardization of processes when conparing in the workplace. the system!
